# Bessel High-Pass Filter Design

This project presents the design and simulation of a Bessel high-pass active filter using LTspice.

## Project Description
The goal of the project was to design and analyze a high-pass filter with Bessel characteristics, focusing on its frequency response and signal behavior. Two active filter topologies were implemented and compared to observe differences in performance.

## Implemented Topologies
- Sallen-Key biquad
- Kerwin–Huelsman–Newcomb biquad

## Analysis Performed
- Transfer characteristics at different frequencies
- Frequency response comparison between the two topologies
- Evaluation of filter behavior in the frequency domain

## Tools Used
- LTspice
